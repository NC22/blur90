Добавить файл \ файлы в Список Воспроизведения (кнопка M) или открыть файл-список можно следующими способами :

- Выбрать его через встроенный Файловый Менеджер (кнопка N, управление описано ниже)
- Открыть файл с помощью -> выбрать bplayer.exe
- Выделить и перетащить файлы на bplayer.exe 
- Через командную строку bplayer.exe [полный путь до файла 1] [полный путь до файла 2] ...

Таким образом можно открыть сразу несколько списков, добавлять \ удалять из них файлы, формируя новый список.

-Дополнительные клавиши стандартные (если у вас расширенная клавиатура):

PLAY\PAUSE      -  Начать воспроизведение \ Пауза
  STOP          -  Остановить воспроизведение
|<<\>>|         -  Предыдущий \ Следующий файл
 - и +          -  Регулирование громкости 
 

Дополнительные клавиши реагируют на нажатие даже если окно программы свернуто \ неактивно.


-Клавиши которые реагируют на нажатия только если окно программы активно:

P               - Начать воспроизведение \ пауза
S               - остановить воспроизведение
< и >           - предыдущий \ следующий файл 
- и +           - регулирование громкости

Управление Списком Воспроизведения (кнопка M) :

  навигация - стрелка вниз , стрелка вверх
  выбор пункта меню - ENTER 
  удалить элемент списка - Delete
  сохранить список в рабочую папку программы - Save


Управление Файловым Менеджером (кнопка N) :

  навигация - стрелка вниз , стрелка вверх
  выбор пункта меню - ENTER (плеер добавит в Список Воспроизведения все выделенные файлы включая текущий)
  выделить \ снять выделение одного файла - Insert  
  выделить все файлы - P 
   

Режимы воспроизведения :

Зациклено - повторяет воспроизведение текущего файла вновь и вновь
Случайно  - выбирает случайный файл из списка воспроизведения
По списку - проигрывает файлы в порядке соответствующем Списку Воспроизведения


Визуальное отображение :

возможности консоли по визуализации крайне ограничены , но есть два варианта
Осцилограмма (параметр в файле настроек DrawMode= 2)
Спектр (параметр в файле настроек DrawMode= 1)

Если окно неактивно - картинка замирает в целях экономии ресурсов 


Локализация :

Программа автоматически открывает первый найденный в рабочей директории файл с расширением *.cplang
Изначально есть файл rus.cplang - Русский язык
Английский язык загружается в случае отсутствия файла локализации


Системный файл настроек : (сохраняет текущие настройки приложения, менять вручную не рекомендуется)

Параметры 
Left,Top - определяют позицию окна, позиция окна сохраняется при выключении приложения
Mode - последний выбранный режим воспроизведения
DrawMode - режим визуального отображения
